# Guia de Contribuição

Obrigado pelo seu interesse em contribuir para o **TV & Cinema Tracker**! Este projeto visa proporcionar a melhor experiência possível para amantes de cinema e séries.

## Como posso contribuir?

### Reportar Bugs
Se encontrar algum erro, por favor abra uma *Issue* descrevendo:
- O comportamento esperado.
- O que aconteceu na realidade.
- Passos para reproduzir o erro.
- Detalhes do dispositivo (versão do Android, etc).

### Sugerir Funcionalidades
Novas ideias são sempre bem-vindas! Abra uma *Issue* com a tag `enhancement`.

### Enviar um Pull Request
1. Faça um **Fork** do repositório.
2. Crie uma branch para a sua alteração: `git checkout -b feature/minha-melhoria`.
3. Garanta que o código segue o padrão do projeto (Tailwind para estilos, React 19 hooks).
4. Faça commit das alterações: `git commit -m "feat: adiciona x funcionalidade"`.
5. Faça push para a branch: `git push origin feature/minha-melhoria`.
6. Abra um **Pull Request**.

## Padrões de Código
- Use **TypeScript** para todo o código novo.
- Utilize componentes funcionais com **React Hooks**.
- Estilize usando **Tailwind CSS**.
- Mantenha a acessibilidade em mente.

Agradecemos a sua ajuda para tornar este projeto melhor!
